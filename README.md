- [react js](https://react.dev/) - JavaScript library for building user interfaces
- [react router](https://reactrouter.com/)- Declarative routing for React applications
- [jotai](https://jotai.org) - Primitive and flexible state management for React
- [tailwind css](https://tailwindcss.com/) - A utility-first CSS framework
- [motion (previously framer-motion)](https://motion.dev) -  A modern animation library for React
- [styled components](https://styled-components.com/) - Visual primitives for the component age
- [phospher icons](https://phosphoricons.com/) - Flexible icon family for interfaces, diagrams, presentations
- [git](https://git-scm.com/) - Distributed version control system
- [vite](https://vite.dev) - Next Generation Frontend Tooling
- [biomejs](https://biomejs.dev/) - A performant formatter, linter, and more for JavaScript, TypeScript, JSX, and JSON
